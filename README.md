# QuickBrandNG

## What
QuickBrandNG is a fast branding studio for small businesses in Nigeria, providing logos, brand kits, and starter websites.

## Why
Helps entrepreneurs quickly launch a consistent brand without design expertise.

## How to run
1. Clone the repo
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
4. Build for production: `npm run build`
